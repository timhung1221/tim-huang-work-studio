References - Windows Desktop App Demo

1) Application Name: Rename

Description: Rename a file name according to need.

Usage:

  • Drop drag a picture file to the form.
  • Then click a Rename-size button.
  • The file name will change to a new file name which from Rule set.
  • The button of Delete can delete a file which selected.
  • Clear button can clear file list.

2) Application Name: Mp3 player

Description: To play mp3 file which you chose. Aim is to learn English.

Usage:

  • Click List File button to chose files directory. All file is mp3 format will be shown in the left list box.
  • Select a file from list and choose a play mode on right. Then click Play button to play mp3.
  • There will be displayed a subtitle if you made it for mp3 file.
  • Also function provided play, pause and stop to play mp3 file.
  • Click Reset button to clear interface if you would like to choose new directory.

3) Application Name: Morse Code Sample

Description: Morse Code is an application to play morse code from you input.

Usage:

  • First step to input some text which to turn morse code.
  • Second click the button of Encode. Then you can hear and see play morse code from the application.

4) Application Name: Password Power

Description: Generate password.

Usage:

Three modes to generate password. You can choose random or memory words or PIN password.

5) Application Name: QR Coder

Description: Generator for QR code.

Usage:

  • Input text first.
  • Then click Encode button to generate QR code.
  • Also, you can add logo picture to generated QR code.
  • This application support decodes QR code also. First load picture then clicks Decode button.

6) Application Name: Binary Tree Seek

Description: A demo for how binary tree run.

Usage:

  • Choice node count and generated number between min to max.
  • Then click Generating button to see binary tree.
  • You can try Save Picture or Find Node function as well.
